## Alert CSS

<Markdown>
{`
Class names are used to style elements. They are also used to select elements in JavaScript.
Use with __sx__ prop or __css__

| Class name          | Description                           |
| ------------------- | ------------------------------------- |
| 'accordion-root'    | The root element of the accordion.    |
| 'accordion-header'  | The header element of the accordion.  |
| 'accordion-content' | The content element of the accordion. |

`}

</Markdown>
