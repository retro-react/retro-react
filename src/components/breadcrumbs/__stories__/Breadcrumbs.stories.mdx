## Breadcrumbs CSS

<Markdown>
{`
Class names are used to style elements. They are also used to select elements in JavaScript.
Use with __sx__ prop or __makeStyles__ function.

| Class name          | Description                                        |
| ------------------- | -------------------------------------------------- |
| 'breadcrumbs-root'  | The root element of the breadcrumbs.               |
| 'breadcrumb'        | The breadcrumb element itself.                     |
| 'breadcrumb-active' | The active breadcrumb item.                        |
| 'breadcrumb-anchor' | The anchor element for non-active breadcrumb item. |

`}

</Markdown>
