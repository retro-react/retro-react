## Marquee CSS

<Markdown>
{`
Class names are used to style elements. They are also used to select elements in JavaScript.
Use with __sx__ prop or __css__

| Class name        | Description                         |
| ----------------- | ----------------------------------- |
| 'marquee-root'    | The root element of the marquee.    |
| 'marquee-content' | The content element of the marquee. |

`}

</Markdown>
