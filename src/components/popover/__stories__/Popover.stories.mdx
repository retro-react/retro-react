## Popover CSS

<Markdown>
{`
Class names are used to style elements. They are also used to select elements in JavaScript.

You an target the components directly with the **sx** prop or use the class names to target the elements in your CSS.
Use with **sx** prop or **css**

| Class name             | Description                              |
| ---------------------- | ---------------------------------------- |
| 'popover-button-root'  | The root element of the popover button.  |
| 'popover-content-root' | The root element of the popover content. |

`}

</Markdown>

## Examples

<Source
	dark
	language="tsx"

    </PopoverContent>
