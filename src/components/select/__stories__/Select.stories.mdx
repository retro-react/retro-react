## Select CSS

<Markdown>
{`
Class names are used to style elements. They are also used to select elements in JavaScript.
Use with __sx__ prop or __css__

| Class name     | Description                     |
| -------------- | ------------------------------- |
| 'select-root'  | The root element of the select  |
| 'select-label' | The label element of the select |
| 'select-input' | The input element of the select |

`}

</Markdown>
