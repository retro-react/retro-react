## Table CSS

<Markdown>
{`
Class names are used to style elements. They are also used to select elements in JavaScript.
Use with __sx__ prop or __css__

| Class name                | Description                                    |
| ------------------------- | ---------------------------------------------- |
| 'table-wrapper'           | The outermost wrapper of the Table component.  |
| 'table-root'              | The root element of the Table component.       |
| 'retro-table-header'      | The container for the table headers.           |
| 'retro-table-header-cell' | Each individual cell of the table header.      |
| 'retro-table-row'         | Each row of the table in table body.           |
| 'retro-table-cell'        | Each individual cell within a table body row.  |
| 'table-pagination'        | The container for the pagination controls.     |
| 'retro-pagination-button' | The buttons used to navigate through pages.    |
| 'retro-pagination-arrow'  | The arrow icons within the pagination buttons. |

`}

</Markdown>
