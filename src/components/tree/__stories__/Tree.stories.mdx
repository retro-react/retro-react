## Tree CSS

<Markdown>
{`
Class names are used to style elements. They are also used to select elements in JavaScript.
Use with __sx__ prop or __css__

| Class name            | Description                                       |
| --------------------- | ------------------------------------------------- |
| 'tree-root'           | The root element of the tree.                     |
| 'tree-node'           | The tree node element.                            |
| 'tree-node-label'     | The label element of the tree node.               |
| 'tree-node-content'   | The content element of the tree node.             |
| 'tree-node-container' | The container element of the tree node.           |
| 'tree-node-children'  | The children container element of the tree node.  |
| 'tree-expanded'       | The expanded state class for tree node container. |

`}

</Markdown>
